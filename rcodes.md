# R Codes

The following lists a set of R programs that enables the user to perform the analyses described in the book. These R programs requires a companion R package [StatMethRank](http://cran.r-project.org/web/packages/StatMethRank/index.html) and two other R packages [pmr](http://cran.r-project.org/web/packages/pmr/index.html) and [mlogit](http://cran.r-project.org/web/packages/mlogit/index.html). In Chapter 9, we may make use of [JAGS](http://mcmc-jags.sourceforge.net/) for efficient Bayesian simulation.
